# Amelia Social

> Stand alone app to allow us to interact with amelia.

![UI](doc/social-app-demo.gif)

## Requirements

[nodejs](https://nodejs.org/en/)

## Installing

```
npm install
```

## Development

Run the local webpack-dev-server on http://localhost:9000/

```
npm run dev
```

## Deployment

Build the current application

```
npm run build
```
